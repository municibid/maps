# Municibid Maps

Interactive visualizations for Municibid.

## Platform Growth Map

Animated map showing the growth of sellers (gold) and bidders (blue) from 2006 to present.

**Live:** https://municibid.github.io/maps/

## Files

- `index.html` - The animated map visualization
- `platform-growth.json` - GeoJSON data with all user locations and dates
